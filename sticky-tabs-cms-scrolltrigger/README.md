# Sticky Tabs CMS with GSAP ScrollTrigger

## Requirements

### Inside the `<head>` tag

```html
<script
  defer
  src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.4/gsap.min.js"
></script>
<script
  defer
  src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.4/ScrollTrigger.min.js"
></script>
```
